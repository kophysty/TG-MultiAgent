# 2026-01-05 - Недельные списки задач + /start приветствие без версии

## Цель

- Сделать недельные диапазоны для списка задач устойчивыми к формулировкам вида "на эту неделю".
- Убрать дубли версии из приветственного сообщения, но сохранить отображение версии по `/start`.

## Изменения

- Недельные фразы (Tasks):
  - распознаются "на этой неделе", "на эту неделю", "на текущую неделю", "на следующей неделе"
  - недельный список включает:
    - задачи с `Due Date` внутри диапазона
    - плюс `Inbox` (без due date можно, остальные без due date нельзя)
  - дополнительно отфильтровано: Inbox задачи с due date вне диапазона не подтягиваются в недельный список

- Приветствие `/start`:
  - первое сообщение: приветствие без версии, можно с вопросом "Чем могу помочь?"
  - второе сообщение: версия в отдельной строке `Версия: vX.Y.Z`
  - `Start` в reply keyboard не уходит в AI pipeline, чтобы не было второго "приветствия от агента" с версией

## Как проверить

1) Нажать `Start` или отправить `/start`.
   - Должно прийти приветствие без версии.
   - Следом должно прийти отдельное сообщение с версией.

2) Сказать или написать:
   - "Покажи мне все задачи на эту неделю"
   - "Покажи задачи на следующей неделе"
   Ожидаемо: список ограничен диапазоном дат недели плюс Inbox, без других задач без due date.

## Заметки

- Диапазоны недели считаются от текущей даты в `TG_TZ` и заканчиваются в следующий понедельник (exclusive).
